## 建置
1. 將專案複製到本地端
    ```bash
    git clone https://github.com/LiKai2507/Tetris.git
    ```

2. 進入專案資料夾
    ```bash
    cd Tetris
    ```
3. 編譯專案
    ```bash
    run build.bat
    ```

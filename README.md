## 建置
1. 將專案複製到本地端
    ```bash
    git clone https://github.com/LiKai2507/Tetris.git
    ```

2. 進入專案資料夾
    ```bash
    cd Tetris
    ```
3. 編譯專案
    ```bash
    run build.bat
    ```

## 玩法說明
+ 遊戲開始後，玩家可以使用鍵盤上的方向鍵來控制方塊的移動。
+ 方塊會從上方掉落，玩家可以使用上鍵來旋轉方塊。
+ 當方塊堆疊到一行時，會消除一行方塊，並獲得分數。
+ 當方塊堆疊到最上方時，遊戲結束。
